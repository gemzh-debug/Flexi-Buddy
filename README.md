# Flexi Buddy 7.0 Professional

Offline-first flexi-time work companion.

## Included in 7.0
- Editable previous-day timeline entries
- Separate before-lunch and after-lunch/evening paid-time totals
- Quick work logging for short email and message tasks
- Customisable favourite actions
- Work modes for home, office, home visits and hybrid days
- Paid travel tracking
- Automatic editable work diary
- Calendar, dashboard, reports, leave and data backup tools

Upload every file in this folder to the root of the GitHub Pages repository.

## 7.0.13
Flexi-sheet suggestions are now anchored to the actual lunch actions: the morning period finishes at the exact Lunch click and works backwards by paid time; the afternoon period starts at the exact Resume click and works forwards by paid time. Paid breaks remain included and unpaid pauses/lunch are excluded.
