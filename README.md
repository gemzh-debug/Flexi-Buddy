# Flexi Buddy

Flexi Buddy is a private, installable web app for tracking an 8-hour working day, breaks, leave and flexi.

## What is included

- Start, resume and finish work
- Paid break, unpaid break and lunch
- First 15 minutes of each paid break counted as paid; excess counted as unpaid
- Running flexi balance
- Full flexi days and part-flexi deductions
- Annual leave, sickness, dependants leave, special leave, part-year days and bank holidays
- Forgotten-time correction
- History, CSV export and JSON backup
- Local on-device storage

## Important

Opening `index.html` directly from the Files app is not enough for full iPhone Home Screen installation. The folder must be hosted on an HTTPS web address.

Simple hosting options include GitHub Pages, Netlify or Cloudflare Pages. Upload the contents of this folder, open the resulting address in Safari, tap Share, then Add to Home Screen.

Your records are stored in Safari on the device/browser used. Download backups regularly from Settings.
